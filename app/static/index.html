FastAPI

FastAPI is a modern, fast (high-performance), web framework for building APIs with Python 3.6+.

It's built on top of Starlette, a lightweight ASGI framework, and offers features like automatic API documentation, code auto-completion, and validation.

FastAPI is also compatible with asynchronous programming, making it suitable for building scalable and high-performance APIs.

FastAPI is actively maintained and developed by the FastAPI community.

Uvicorn

Uvicorn is a lightning-fast ASGI server, using Python's standard library's asynchronous I/O capabilities.

Uvicorn is known for its high performance, low memory usage, and excellent support for asynchronous applications.

Starlette

Starlette is a lightweight ASGI framework/toolkit, which is ideal for building high-performance asyncio services.

Starlette is designed to be used either as a complete framework, or as an ASGI toolkit that can be used with other ASGI frameworks.

Pydantic

Pydantic is a data validation and settings management library, which is used to validate the data sent to your API.

Pydantic is used to define the data types of your API's request and response data, and to validate that the data is correct.

SQLAlchemy

SQLAlchemy is a SQL toolkit and Object-Relational Mapping (ORM) library for Python.

SQLAlchemy is used to interact with databases in Python, and to map Python objects to database tables.
